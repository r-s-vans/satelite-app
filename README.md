Earth Data Explorer

「衛星データと研究者の知見をシームレスにつなぐ」をコンセプトにした、地図ベースのコラボレーションツールです。JAXA Earth APIを活用し、任意の場所・日時の衛星画像をWebブラウザ上で直感的に可視化。
